## Features
- Home: list of movies (`ListView.builder`, `MovieCard`)
- Detail: poster, title, year, genre, description, favorite toggle
- Favorites: only saved movies
- Profile: static info + bio
- Drawer: Home / About / Contact (from any page)
- BottomNavigationBar: Home / Favorites / Profile

## Run
1. Open the folder in Android Studio or VS Codes.
2. Run `flutter pub get`.
3. Start an emulator or connect a device.
4. Press **Run**.

## Folder Structure
